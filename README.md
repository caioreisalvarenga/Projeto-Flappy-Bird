# Projeto-Flappy-Bird

Esse projeto é uma espécie de Flappy Bird clone, só que pra web, com algumas funcionalidades adicionadas, como por exemplo, quando o usuário morre<br>
mostra na tela que ele morreu e mostra a quantidade de pontos que ele fez, e aí depois, a página atualiza sozinha.

## Tencologias Usadas

Front End: HTML5, CSS3, JavaScript puro.

## Funcionamento

O jogo funciona da seguinte maneira: o jogo é iniciado, e a partir daí vai contando os pontos, e quando o jogador morre, mostra na tela que ele morreu e fala <br>
quantos pontos ele fez.

## Como executar o projeto
  1° - Baixe os arquivos zipados |
  2° - Extraia os arquivos | 
  3° - Execute o flappy.html
